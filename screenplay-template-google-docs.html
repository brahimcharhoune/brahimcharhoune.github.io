Free Screenplay Template for Google Docs (Copy and Use)

Writing a screenplay in Google Docs is simple but formatting it correctly can take time. Many writers struggle with margins, dialogue alignment, and scene structure. In this article, you’ll get a free screenplay template for Google Docs that you can copy and use immediately, along with tips to customize it for your script.

Why Use a Screenplay Template?

A template saves time and ensures your screenplay follows industry standards from the start.

Benefits of Using a Template

  • No need to set margins manually
  • Consistent formatting across your script
  • Faster writing workflow
  • Fewer formatting mistakes

Instead of configuring everything from scratch, you can start writing right away.

Standard Screenplay Template Settings

Before using any template, make sure it follows these professional standards:

  • Font: Courier Prime, 12ptItem two

  • Margins:

    • Top: 1 inch
    • Bottom: 1 inch
    • Left: 1.5 inches
    • Right: 1 inch
  • Page size: US Letter

These settings ensure your script meets industry expectations.

Basic Screenplay Template Structure

Here’s a simple structure you can copy into your Google Docs file:

INT. LOCATION — DAY


Action description goes here. Describe what is happening on screen in present tense.


          CHARACTER NAME

     Dialogue goes here, properly centered and formatted.


          CHARACTER NAME

     (quietly)

     Additional dialogue goes here.


CUT TO:

You can duplicate this structure for each new scene.

How to Use This Template in Google Docs

  • Open a new document in Google Docs
  • Copy and paste the template structure
  • Open CineFormat AI from the Extensions  menu 
  • Click Auto Format Document to automatically apply professional screenplay formatting
  • Review the formatting and make any minor adjustments if needed
  • Duplicate sections for additional scenes

Limitations of Manual Templates

While templates are helpful, they have limitations:

  • Formatting can break when editing
  • Copy-paste can affect alignment
  • Scene numbering must be managed manually
  • Maintaining consistency takes effort

This is why templates alone are not always enough for long scripts.

Tips for Using Screenplay Templates Effectively

  • Keep your template simple and clean
  • Avoid adding unnecessary formatting styles
  • Test formatting before writing a full script
  • Review layout regularly as you write
